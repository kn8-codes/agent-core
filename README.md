# agent-core

Design documents and character specs for the kn8 agent infrastructure.

## What this is

This repo contains **specs and design docs only** — no runnable code. It defines the personality layers, room architecture, and agent interaction patterns used across the fleet.

- `AGENT_CHARACTER.md` — personality matrix and voice definitions
- `ROOMS-DESIGN-SPEC-v1.md` — room-based agent environment architecture
